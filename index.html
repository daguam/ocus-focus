What is OcusFocus?

Ocus Focus is my first web project focused on learning the basics while trying to understand what is really happening behind the scenes, hopefully to save me some trouble on the long run. This project also looks to dive in some of the technologies involved in the development process and explore different points of view in which are the best practices used in the web development community and why. This project is mainly based on MDN web docs' web development tutorial and some more of their references and guides.

Checkout the Github project here

The focus is on:

  • Structure (Html)
  • Style (CSS)
  • Dynamic client-side scripting (JavaScript)
  • Tools and testing
  • Git and Github
  • Server-side website programming
  • Accessibility
  • Graphich Design

The main goal is to develop the skills needed to work with other people and create awesome projects.
